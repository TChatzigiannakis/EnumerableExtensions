README

What is this repository for?

  • Source code for EnumerableExtensions, a portable class library (PCL) offering extension methods for IEnumerable in C#, mostly following the spirit of the existing LINQ methods (and overloading them wherever applicable).
  • NUnit test project.

How do I get set up?

  • Clone this repository.
  • Open the solution file using a version of Visual Studio that supports the PCL project type (2013 Ultimate, for example) or using Xamarin Studio (free from the MonoDevelop website).
